# MACD
利用backtrader框架进行了回测（20180520-20220920），
MACD技术指标不同周期交易EOS/ETH的收益表现，
回测代码见MACD文件，
收益的具体表现见 EOS/ETH文件夹内的html文件。
为制定科学的量化策略提供一些可参考的启发。

通过收益表现，可得出一些结论：
1 交易周期越大 MACD的可靠性越高
2 纯依靠MACD进行交易 不如囤币啥也不干
3 根据MACD交易 ETH的收益表现大幅领先EOS
